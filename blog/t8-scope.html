Scope

What 'scope' is, and how does it work in JavaScript?

Scope in Javascript is about context. There is 'Global' and 'Local' scope. If you have a piece of code, then that piece of code will only ever have one global scope, but it can have multiple local scopes. What is defined in the global scope, for example a variable, can be used through that whole piece of code. Local scope is more limited, it can only be used in a particular piece of contained code. An example of this is a variable within a function. This variable is not accessible outside of that function.
